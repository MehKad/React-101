What's React

react is a JavaScript library created by Facebook, Chocking lol.
It's a User Interface library (UI), a tool for building it's components

How can I add react to my website?

  1. Load React API

    2. <script src= "https://unpkg.com/react@16/umd/react.production.min.js"></script>

  2. Load React DOM

    3. <script src= "https://unpkg.com/react-dom@16/umd/react-dom.production.min.js"></script>

  3. Load Babel Compiler

    4. <script src="https://unpkg.com/babel-standalone@6.15.0/babel.min.js"></script>

  4. Babel Code

    5. <script type="text/babel"> JSX Babel code goes here </script>

Of-course, you're gonna be wondering what the fuck is Babel?

Thats me btw!!!

Well I'm here to answer that.

Babel is a Javascript compiler that can translate markup or programming languages into JS, with Babel you can use the newest features of JS like (ES6-ECMAScript 2015)
