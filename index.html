API

API stands for Application Programming Interface, it's an application programming interface for the web, it can extend the functionality of a web browser
It's a way for two or more computer programs to communicate with each other


A famous example of the usage of APIs

The log-in using Facebook/Twitter/Google/Github... functionality you see on so many websites. It’s incredibly convenient, well that's using APIs, and you might be wondering how does it work, here's how
The way it works is pretty simple. Every time the application loads, it uses the API to check whether the user is already logged in by means of whatever social media platform. If not, when the user clicks the “Log-in Using XYZ” button, a pop-up opens where they are asked to confirm they actually want to log-in with that social media profile. When the user confirms, the API provides the application with identification information, so it knows who’s logging in.

Babel is a Javascript compiler that can translate markup or programming languages into JS, with Babel you can use the newest features of JS like (ES6-ECMAScript 2015)

Okay, so we all like new stuff, the iphone 14, the newest mac and stuff, well Babel got you, Developpers like you (I guess you are one) can use new JavaScript language features by using Babel to convert their source code into a version of JavaScript that a webbrowser can process.
